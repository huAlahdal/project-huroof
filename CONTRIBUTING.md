# المساهمة في مشروع حروف

نرحب بجميع المساهمات في مشروع حروف! سواء كنت تريد إصلاح خطأ، إضافة ميزة جديدة، أو تحسين الوثائق، فنحن سعداء بوجودك معنا.

## 🤝 كيفية المساهمة

### 1. Fork المشروع
- اذهب إلى صفحة المشروع على GitHub
- انقر على زر "Fork"
- انسخ المشروع إلى حسابك

### 2. إعداد البيئة المحلية
```bash
# استنسخ المشروع
git clone https://github.com/YOUR_USERNAME/huroof.git
cd huroof

# إعداد الواجهة الخلفية
cd backend
dotnet restore
dotnet ef database update

# إعداد الواجهة الأمامية
cd ../frontend
npm install
```

### 3. إنشاء فرع جديد
```bash
git checkout -b feature/اسم-الميزة
# أو
git checkout -b fix/إصلاح-الخطأ
```

### 4. إجراء التغييرات
- قم بإجراء التغييرات المطلوبة
- أضف اختبارات إذا لزم الأمر
- تأكد من أن الكود يعمل بشكل صحيح

### 5. Commit التغييرات
```bash
git add .
git commit -m "وصف واضح للتغييرات"
```

### 6. الدفع وإنشاء Pull Request
```bash
git push origin feature/اسم-الميزة
```
- اذهب إلى GitHub وإنشاء Pull Request
- اشرح التغييرات التي قمت بها
- انتظر المراجعة

## 📝 قواعد المساهمة

### الكود
- استخدم أسماء متغيرات واضحة باللغة الإنجليزية
- أضف تعليقات للكود المعقد
- احترام نمط الكود الحالي
- تأكد من أن الكود يعمل على جميع المتصفحات الحديثة

### الرسائل (Commit Messages)
- استخدم اللغة الإنجليزية في رسائل commit
- اجعل الرسالة واضحة ومختصرة
- مثال: `Add buzzer sound effects` أو `Fix timer display issue`

### الوثائق
- حدث الوثائق عند إضافة ميزات جديدة
- استخدم اللغة العربية في الوثائق للمستخدمين
- استخدم اللغة الإنجليزية في الكود والتعليقات التقنية

## 🐛 الإبلاغ عن المشاكل

عند الإبلاغ عن مشكلة:
1. تحقق من أن المشكلة لم تبلغ عنها من قبل
2. استخدم قالب الإبلاغ عن المشاكل
3. أضف تفاصيل كافية:
   - نظام التشغيل والمتصفح
   - خطوات إعادة المشكلة
   - لقطات شاشة إذا أمكن
   - رسائل الخطأ

## 💡 اقتراحات الميزات

عند اقتراح ميزة جديدة:
1. تحقق من أن الميزة لم تطلب من قبل
2. اشرح لماذا هذه الميزة مهمة
3. صف كيف ستعمل
4. فكر في التأثير على المستخدمين الحاليين

## 🎨 أنواع المساهمات المطلوبة

### تطوير
- إصلاح الأخطاء

### تصميم

- إضافة رسوم متحركة
- تحسين التصميم المتجاوب
- إنشاء أيقونات أو رسومات

### محتوى
- إضافة أسئلة جديدة
- تحسين الأسئلة الحالية
- ترجمة المحتوى
- كتابة وثائق

### اختبار
- كتابة اختبارات تلقائية
- اختبار على متصفحات مختلفة
- اختبار على أجهزة مختلفة
- الإبلاغ عن الأخطاء
