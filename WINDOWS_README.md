# 🎮 تشغيل حروف على ويندوز للعب مع الأصدقاء

دليل سريع لتشغيل لعبة حروف على جهازك ويندوز واللعب مع أصدقائك على الشبكة المحلية.

---

## 📋 المتطلبات الأساسية

قبل البدء، تأكد من تثبيت البرامج التالية:

1. **.NET 10.0** 
   - رابط التحميل: https://dotnet.microsoft.com/download
   - اختر .NET 10.0 SDK

2. **Node.js**
   - رابط التحميل: https://nodejs.org
   - اختر النسخة LTS (مستقرة)

---

## 🚀 طرق التشغيل

### الطريقة الأولى: النص البرمجي التلقائي (موصى به)

1. افتح مجلد المشروع
2. انقر نقراً مزدوجاً على `deploy-windows.bat`
3. انتظر حتى تظهر رسالة النجاح
4. انسخ الرابط الذي يظهر وأرسله لأصدقائك

### الطريقة الثانية: PowerShell

1. انقر بزر الماوس الأيمن على `deploy-windows.ps1`
2. اختر "Run with PowerShell"
3. إذا ظهرت رسالة أمان، اختر "Run anyway"
4. انتظر حتى يظهر الرابط

### الطريقة الثالثة: التشغيل اليدوي

#### الخطوة 1: تشغيل الواجهة الخلفية
```cmd
cd backend
dotnet run
```
- انتظر حتى تظهر رسالة: `Now listening on: http://localhost:5062`

#### الخطوة 2: تشغيل الواجهة الأمامية
- افتح موجه أوامر جديد
```cmd
cd frontend
npm install
npm run dev
```
- انتظر حتى يظهر: `Local: http://localhost:5173`

---

## 🌐 السماح للأصدقاء بالانضمام

### 1. الحصول على عنوان IP
عند تشغيل اللعبة، سيظهر لك عنوان IP مثل:
```
http://192.168.1.100:5173
```

### 2. فتح المنفذ في جدار الحماية
إذا لم يتمكن الأصدقاء من الاتصال:

1. ابحث عن "Windows Defender Firewall" في قائمة ابدأ
2. اختر "Allow an app through firewall"
3. انقر "Change settings"
4. انقر "Allow another app..."
5. اختر "Browse" وحدد مسار Node.js
6. تأكد من تحديد "Private" و "Public"
7. انقر "OK"

---

## 📱 كيف ينضم الأصدقاء

1. يفتحون الرابط الذي أرسلته في متصفحهم
2. يكتبون اسمهم
3. يختارون دورهم:
   - الفريق البرتقالي 🟠
   - الفريق الأخضر 🟢
   - مدير اللعبة 🎮 (لشخص واحد فقط)
   - مشاهد 👀

---

## 🔧 حل المشاكل الشائعة

### ❌ "الأصدقاء لا يستطيعون الاتصال"
**الحلول:**
- تأكد أن الجميع على نفس شبكة Wi-Fi
- تحقق من جدار الحماية (انظر أعلاه)
- جلب الرابط الصحيح مع IP الخاص بك

### ❌ "خطأ .NET"
**الحل:**
- أعد تشغيل الكمبيوتر بعد تثبيت .NET
- تأكد من تثبيت نسخة 10.0 تحديداً

### ❌ "خطأ Node.js"
**الحل:**
- شغّل موجه الأوامر كـ Administrator
- تأكد من تثبيت النسخة 18 أو أحدث

### ❌ "المنفذ مستخدم"
**الحل:**
- أغلق جميع البرامج التي تستخدم المنفذ 5173
- أعد تشغيل الكمبيوتر وحاول مرة أخرى

---

## 🎮 نصائح للعب الممتعة

1. **استخدم تطبيق صوتي** مثل Discord أو WhatsApp للتواصل
2. **اتصل بالإنترنت عبر كابل** إذا كانت Wi-Fi بطيئة
3. **مدير اللعبة فقط** يمكنه بدء اللعبة وتغيير الإعدادات
4. **يمكن تغيير الفرق** أثناء اللعبة إذا لزم الأمر

---

## 🔄 إعادة التشغيل والإيقاف

### لإيقاف اللعبة:
- أغلق نوافذ موجه الأوامر
- أو اضغط `Ctrl+C` في كل نافذة

### لإعادة التشغيل:
- شغّل `deploy-windows.bat` مرة أخرى

## 📞 المساعدة

1. تحقق من المتطلبات الأساسية
2. تأكد من تحديث ويندوز
3. جرب التشغيل كـ Administrator
4. أعد تشغيل جهاز الكمبيوتر
