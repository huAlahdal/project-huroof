# 🎮 تشغيل حروف على ويندوز للعب مع الأصدقاء

## 📋 المتطلبات

1. **تثبيت .NET 10.0**
   - اذهب إلى: https://dotnet.microsoft.com/download
   - قم بتنزيل وتثبيت .NET 10.0 SDK

2. **تثبيت Node.js**
   - اذهب إلى: https://nodejs.org
   - قم بتنزيل وتثبيت النسخة LTS

## 🚀 طريقة التشغيل السريعة

### الطريقة الأولى: استخدام النص البرمجي (موصى به)

1. افتح مجلد المشروع
2. انقر نقراً مزدوجاً على `deploy-windows.bat`
3. انتظر until يظهر عنوان IP الخاص بك
4. أرسل العنوان لأصدقائك

### الطريقة الثانية: التشغيل اليدوي

#### 1. تشغيل الواجهة الخلفية (Backend)
```cmd
cd backend
dotnet run
```
- انتظر until ترى رسالة مثل: `Now listening on: http://localhost:5062`

#### 2. تشغيل الواجهة الأمامية (Frontend)
- افتح موجه أوامر جديد
```cmd
cd frontend
npm install
npm run dev
```
- انتظر until ترى: `Local: http://localhost:5173`

## 🌐 السماح للأصدقاء بالانضمام

### 1. معرفة عنوان IP الخاص بك
- افتح موجه الأوامر واكتب:
```cmd
ipconfig
```
- ابحث عن "IPv4 Address" تحت محول الشبكة الخاص بك
- سيبدو مثل: `192.168.1.100`

### 2. فتح المنفذ في جدار الحماية
- افتح "Windows Defender Firewall with Advanced Security"
- اذهب إلى "Inbound Rules"
- انقر "New Rule..."
- اختر "Port"
- اختر "TCP" واكتب `5173`
- اختر "Allow the connection"
- حدد جميع الملفات الشخصية (Domain, Private, Public)
- سمِّ القاعدة "Huroof Game"
- انقر "Finish"

### 3. إرسال الرابط لأصدقائك
أرسل لهم الرابط التالي:
```
http://YOUR_IP_ADDRESS:5173
```
مثال:
```
http://192.168.1.100:5173
```

## 🔧 حل المشاكل الشائعة

### المشكلة: الأصدقاء لا يستطيعون الاتصال
**الحل:**
1. تأكد من أن جميع الأجهزة على نفس الشبكة (Wi-Fi)
2. تحقق من جدار الحماية
3. جرب تعطيل جدار الحماية مؤقتاً للاختبار

### المشكلة: خطأ في .NET
**الحل:**
1. تأكد من تثبيت .NET 10.0 SDK
2. أعد تشغيل الكمبيوتر بعد التثبيت

### المشكلة: خطأ في Node.js
**الحل:**
1. تأكد من تثبيت Node.js 18 أو أحدث
2. افتح موجه أوامر جديد كـ Administrator

### المشكلة: الواجهة الخلفية لا تعمل
**الحل:**
1. انتظر until تظهر رسالة الاستماع
2. تحقق من أن المنفذ 5062 غير مستخدم

## 📱 كيف يلعب أصدقاؤك

1. يفتحون الرابط الذي أرسلته في متصفحهم
2. يكتبون أسماءهم
3. ينضمون إلى الغرفة التي أنشأتها
4. يختارون الفريق (برتقالي أو أخضر)

## 🎮 نصائح للعب الجماعي

1. **استخدم Discord أو WhatsApp** للتواصل الصوتي
2. **تأكد من اتصال الإنترنت جيد** للجميع
3. **يمكن لمدير اللعبة التحكم الكامل** في اللعبة
4. **يمكن طرد اللاعبين المزعجين** إذا لزم الأمر

## 🔄 إعادة التشغيل

لإيقاف الخوادم:
- أغلق نوافذ موجه الأوامر

لإعادة التشغيل:
- شغّل `deploy-windows.bat` مرة أخرى

إذا واجهت أي مشاكل:
1. تحقق من المتطلبات
2. تأكد من أن جميع الملفات موجودة
3. جرب التشغيل كـ Administrator
